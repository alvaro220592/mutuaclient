listar dispositivos: adb devices
iniciar o servidor dev android: quasar dev -m capacitor -T android
rodar o app sem android studio: npx cap run android
rodar o app com android studio: android-studio android
rodar no emulador:
    quasar dev -m capacitor -T android
    emulator -list-avds
    emulator -avd <NomeDoEmuladorListado ex: Pixel_7>

variáveis de ambiente devem começar com 'VITE_', senão o quasar bloqueia

instalar plugin google auth(login com google): npm install @capawesome/capacitor-google-sign-in

sempre que instalar um plugin pro android: npx cap sync android

limpar cache:
    rm -rf node_modules/.q-cache
    rm -rf android/.gradle
    rm -rf android/app/build
    npx cap sync android
    cd android
    ./gradlew clean
    quasar dev -m capacitor -T android

    no android studio:
        File
        → Invalidate Caches
        → Invalidate and Restart

buildar para casos em que plugins não funcionam no servidor dev, como o GoogleSignIn:
    quasar build -m capacitor -T android
    npx cap sync android
    #npx cap run android # pelo terminal (sem android studio)
    android-studio android # pelo andoid studio

colocar em produção na playstore
    Criar keystore:
        keytool -genkeypair -v -keystore mutua-release.keystore -keyalg RSA -keysize 2048 -validity 10000 -alias mutua
        ele salva no diretorio atual numa pasta chamada mutua-release.keystore. Tem que colocar essa pasta no gitignore
    pegar SHA1 dele:
        keytool -list -v -keystore mutua-release.keystore -alias mutua
    cria um novo id de cliente Oauth na google cloud
    coloca lá o sha1 criado
    muda a var de ambiente VITE_GOOGLE_AUTH_ENABLED para true
    builda: quasar build -m capacitor -T android
    sincroniza: npx cap sync android (pode entrar em src-capacitor e fazer o mesmo comando por via das dúvidas)
    gera o bundle:
        cd android
        ./gradlew bundleRelease
        vai ser gerado em android/app/build/outputs/bundle/release/app-release.aab
    Como AAB não instala direto, gere APK local para teste:
        cd android
        ./gradlew assembleRelease
        depois procura: find android -name "*.apk"
    procure: find android -name "*.aab"
        esperado: android/app/build/outputs/bundle/release/app-release.aab
        subir esse arquivo no google play console (https://developer.android.com/distribute/console?hl=pt-br)
